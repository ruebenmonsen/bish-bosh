Bish-Bosh 2.0

What is Bish-Bosh?

Bish-Bosh 2.0 loops through numbers, from 1 to the chosen loop number (field 1). If the number is divisible by the first division number (field 2), Bish will be the result instead of the number. If the number is divisible by the second division number (field 3), Bosh will be the result instead of the number. If the number is divisible by both division numbers, Bish-Bosh will be the result. Otherwise the number itself will be the result.

The result will then be presented in the textarea (the output field).
